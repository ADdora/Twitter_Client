Detailed Description

Created by ahmedadel on 25/06/16.

TwitterApiClient is an customize version of real TwitterApiClient that exists inside Twitter SDK. We create a customize one to handle edit GsonBuilder and add our custom registerTypeAdapter. We have a problem with Retrofit and Gson.Gson doesnâ€™t know that RealmList ist just a wrapper class. So that's why we made a our custom JsonSerializer and JsonDeserializer for User and Tweets Model. Because these two models are used inside Followers and UserTweets Models as List so they couldn't be casted into normal List<> class
